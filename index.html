Polityka prywatności

Niniejsza polityka prywatności dotyczy aplikacji służącej do automatycznego publikowania postów na fanpage za pomocą API Facebooka. Aplikacja działa wyłącznie na podstawie dostarczonego tokena dostępu i nie zbiera żadnych danych osobowych użytkowników.

1. Zakres przetwarzanych danych

Aplikacja przetwarza wyłącznie token dostępu do fanpage, który umożliwia publikowanie postów. Aplikacja:

  • Nie przechowuje danych osobowych użytkowników.
  • Nie uzyskuje dostępu do danych logowania ani danych konta użytkownika.

2. Cel przetwarzania danych

Token dostępu jest wykorzystywany wyłącznie w celu:

  • Publikacji postów na fanpage zgodnie z konfiguracją użytkownika.

3. Bezpieczeństwo danych

Aplikacja korzysta z bezpiecznego połączenia HTTPS do komunikacji z API Facebooka. Token dostępu jest przechowywany i przetwarzany w bezpieczny sposób, zapewniając ochronę przed nieautoryzowanym dostępem.
