Polityka prywatności

Niniejsza polityka prywatności dotyczy aplikacji służącej do automatycznego publikowania postów na fanpage za pomocą API Facebooka. Aplikacja nie zbiera ani nie przechowuje danych osobowych użytkowników poza informacjami niezbędnymi do autoryzacji i publikacji postów.

1. Jakie dane są zbierane

Aplikacja korzysta z API Facebooka, które umożliwia:

  • Dostęp do fanpage, na którym publikowane są posty.
  • Dostęp do podstawowych informacji o użytkowniku (takich jak imię i nazwisko), wymagany wyłącznie w celu autoryzacji.

Aplikacja nie przechowuje żadnych danych osobowych – dane te są przetwarzane wyłącznie podczas sesji użytkownika w celu publikowania treści.

2. W jaki sposób dane są wykorzystywane

Dane są wykorzystywane wyłącznie do:

  • Autoryzacji użytkownika za pomocą Facebook Graph API.
  • Publikacji postów na fanpage zgodnie z instrukcją użytkownika.

3. Udostępnianie danych

Aplikacja nie udostępnia żadnych danych osobowych stronom trzecim.

4. Bezpieczeństwo danych

Aplikacja korzysta z bezpiecznego połączenia HTTPS oraz mechanizmów autoryzacji dostarczanych przez Facebook w celu ochrony danych podczas komunikacji z API.
